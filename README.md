# 🚀 Portfolio 3D - Adem Yacef

Un portfolio moderne et interactif développé avec React, TypeScript et Three.js, présentant mes compétences en développement web et mobile.

## ✨ Fonctionnalités

- 🎨 **Design 3D Interactif** - Animations 3D avec Three.js
- 📱 **Responsive Design** - Optimisé pour tous les appareils
- ⚡ **Performance** - Construit avec Vite pour des temps de chargement rapides
- 🎯 **Modern Stack** - React 19, TypeScript, Tailwind CSS
- 🔧 **UI Components** - Composants Radix UI pour une expérience utilisateur premium

## 🛠️ Technologies Utilisées

### Frontend
- **React 19** - Bibliothèque UI moderne
- **TypeScript** - Typage statique pour une meilleure robustesse
- **Vite** - Outil de build ultra-rapide
- **Tailwind CSS** - Framework CSS utility-first
- **Three.js** - Rendu 3D et animations

### UI/UX
- **Radix UI** - Composants accessibles et personnalisables
- **Framer Motion** - Animations fluides
- **Lucide React** - Icônes modernes

### Déploiement
- **GitHub Pages** - Hébergement gratuit
- **GitHub Actions** - Déploiement automatique

## 🚀 Déploiement

Le site est automatiquement déployé sur GitHub Pages à chaque push sur la branche `main`.

**🌐 Site en ligne :** [https://adem-yac.github.io/Adem-Yac/](https://adem-yac.github.io/Adem-Yac/)

## 📦 Installation Locale

```bash
# Cloner le repository
git clone https://github.com/Adem-yac/Adem-Yac.git

# Naviguer vers le dossier
cd Adem-Yac

# Installer les dépendances
npm install

# Lancer le serveur de développement
npm run dev
```

## 🎯 Scripts Disponibles

```bash
npm run dev          # Serveur de développement
npm run build        # Build de production
npm run preview      # Aperçu du build
npm run lint         # Vérification du code
npm run deploy       # Déploiement manuel
```

## 🔧 Configuration

### Vite
Le projet utilise Vite avec la configuration suivante :
- Base path : `/Adem-Yac/` (pour GitHub Pages)
- Plugin React
- Alias `@` pour `./src`

### GitHub Pages
- Déploiement automatique via GitHub Actions
- Configuration dans `.github/workflows/deploy.yml`
