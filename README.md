# 🚀 Portfolio 3D - Adem Yacef

Un portfolio moderne et interactif développé avec React, TypeScript et Three.js, présentant mes compétences en développement web et mobile.

## ✨ Fonctionnalités

- 🎨 **Design 3D Interactif** - Animations 3D avec Three.js
- 📱 **Responsive Design** - Optimisé pour tous les appareils
- ⚡ **Performance** - Construit avec Vite pour des temps de chargement rapides
- 🎯 **Modern Stack** - React 19, TypeScript, Tailwind CSS

## 🛠️ Technologies Utilisées

### Frontend
- **React 19** - Bibliothèque UI moderne
- **TypeScript** - Typage statique pour une meilleure robustesse
- **Vite** - Outil de build ultra-rapide
- **Tailwind CSS** - Framework CSS utility-first
- **Three.js** - Rendu 3D et animations

### UI/UX
- **Radix UI** - Composants accessibles et personnalisables
- **Framer Motion** - Animations fluides
- **Lucide React** - Icônes modernes
